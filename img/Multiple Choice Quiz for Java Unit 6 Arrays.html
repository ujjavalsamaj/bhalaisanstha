 Multiple Choice Java Array Quiz
 

Directions:  Choose the best answer.  Use the button at the bottom
to check your answers.

1.  Which of the following declares an array of integers named number?
     [a] int number ;
     [b] int [ ] number ;
     [c] int new number [ ] ;
     [d] int number = int [ ] ;
 

2.  What is the output of the following code fragments?
     int [ ] fun = new int [5];
     fun[0] = 1;
     fun[1] = 2;
     fun[2] = 3;
     fun[3] = 4;
     fun[4] = 5;
     int j = 3;
     System.out.println(fun[ j-1]) ;

     [a] 1
     [b] 2
     [c] 3
     [d] 4

3.  What is the output of the following code fragment?
     int [ ] odd = {1, 3, 5, 7, 9, 11 };
     System.out.println( odd[0] + "  " + odd[3] ) ;

     [a] 1  5
     [b] 6
     [c] 1  7
     [d] 8

4.  What does the following method do?
     public static void numbers (int [ ] num)
    {
          for(int x = 1; x < num.length; x++)
              num[x] = num[0];
     }          

     [a] It copies what is in cell 0 into all other cells.
     [b] It changes every element to the value of  x.
     [c] It copies 0 into every element.
     [d] It puts the array into numerical ascending order.

8.  What is the output of the following code fragment?

     [d] 2  7  3  5  8  9

17.  What does this method do?
        public static void cutUp(String word, char [ ] letter)
        {
               if (letter.length >= word.length( ) )
                  for (int i = 0; i < word.length( ); i++)
                    letter[i] = word.charAt(i);
         }

     [a] It checks to see if the array contains the same letters as the String.
     [b] It copies letters from the array to the String.
     [c] If there are enough cells in the array, it copies each letter from the String
                to the array.
     [d] It creates a new array that holds the letters from the String.
